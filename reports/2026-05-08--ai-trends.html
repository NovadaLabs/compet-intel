Claude Opus 4.7 GA + Managed Agents "Dreaming" — Self-Improving Long-Running Agents Arrive

Anthropic released Claude Opus 4.7 (GA) with major software engineering gains. Managed Agents ships three new capabilities: Multi-agent orchestration (public beta), Outcomes — iterate until success criteria met (public beta), and "Dreaming" — agents inspect prior session history and self-improve (research preview). Backed by the Anthropic–SpaceX Colossus deal: 300 MW, 220,000+ NVIDIA GPUs online within the month.

Anthropic 发布 Claude Opus 4.7（正式版），在高难度软件工程任务上取得重大进展。Managed Agents 新增三大能力：多智能体编排（公测），结果导向——自动迭代直至满足成功标准（公测），"梦境"——智能体审查历史会话自我改进（研究预览）。背后是 Anthropic 与 SpaceX Colossus 的算力协议：300 兆瓦、超 22 万块英伟达 GPU 于本月上线。

Novada Angle

Position as "built for agentic access patterns, not just human browsing." 智能体循环审查历史（"梦境"）会产生与人类浏览截然不同的突发、快速重复网络数据请求。Novada 的代理轮换和速率限制管理是直接瓶颈。定位为"专为智能体访问模式设计，而非仅供人类浏览"。

GPT-5.5 Instant 与 Claude Opus 4.7 同周发布——多模型管道成为企业默认选择

OpenAI 的 GPT-5.5 Instant（快速推理，GPT-5 系列）与 Claude Opus 4.7 在4月30日至5月7日同周发布，同时 Gemini 多模态 RAG 也加速落地。企业团队现已习惯在单一管道中跨 Claude、GPT、Gemini 路由任务。Claude Code 将 Pro/Max/Team 速率限制翻倍，取消高峰时段限制，新增插件 URL 加载——明确为高吞吐量智能体工作负载提供支持。

Novada Angle
Multi-model pipelines send web data requests from increasingly diverse SDKs. Framework-agnostic, model-agnostic proxy APIs become more valuable. Novada should publish compatibility guides for all three major provider SDKs and ensure the MCP server works with Claude Code's new plugin URL loading. 多模型管道从越来越多样的 SDK 发送网络数据请求，框架无关、模型无关的代理 API 价值提升。Novada 应为三大主流提供商 SDK 发布兼容指南，并确保 MCP 服务器与 Claude Code 新插件 URL 加载功能兼容。
