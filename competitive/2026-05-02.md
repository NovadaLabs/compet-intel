## AI Insights

1. **Agentic search is becoming infrastructure, not a feature.** Nebius's $275–400 M acquisition of Tavily signals that real-time search is an expected primitive in any AI cloud stack. Standalone search API vendors without a broader platform will face margin pressure as hyperscalers bundle search natively.

2. **Scrapers are evolving into research agents.** Firecrawl's Spark model family and `/agent` endpoint, Exa's three-tier search system, and Oxylabs' self-healing parsers all point to the same shift: the market is moving from raw-HTML extraction toward intelligent, outcome-oriented data retrieval. Products that only return unstructured content will commoditize.

3. **Open-source carries real enterprise security risk.** Crawl4AI's v0.8.6 supply-chain compromise (litellm) is a textbook illustration. As procurement teams evaluate self-hosted crawlers against SaaS alternatives, CVE history and dependency hygiene will become selection criteria — a window for vendors with strong security posture to differentiate.
