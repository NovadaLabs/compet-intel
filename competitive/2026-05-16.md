## AI Insights

### 1. Firecrawl is engineering toward zero marginal cost per query
The `question` and `highlights` formats using up to 100× fewer tokens, combined with Lockdown Mode (cache-only, no outbound requests), suggest Firecrawl is building a tiered cost architecture where the cheapest queries approach negligible cost. This is a structural attack on the per-scrape pricing model that most competitors — including services Novada may use or compete with — rely on. If Firecrawl can anchor developer expectations around "good enough" token-efficient answers, full-page scraping becomes a premium edge case. Novada should track whether these formats achieve acceptable accuracy, and consider whether its own pipeline can offer similarly efficient retrieval modes.

### 2. Platform distribution is now a first-order competitive variable
Firecrawl is simultaneously embedded in n8n Cloud, OpenRouter, Cursor Marketplace, Docker MCP Catalog, and the Claude Code / Codex / OpenCode skill registries. For most developers, the tool they discover first in their IDE or workflow platform is the tool they adopt. Apify's Agent Skills strategy targets the same vector. Novada's product should be auditable against a concrete distribution checklist: Cursor Marketplace, Claude Desktop MCP config, n8n node, OpenRouter provider, Docker MCP Catalog. Absence from these surfaces is not neutral — it is actively losing developer mindshare to competitors who are present.

### 3. BrightData's SDK-in-TV-apps model faces a platform governance reckoning
BrightData's 150M-node residential proxy network is partly sustained by SDKs embedded in consumer TV apps. Google and Amazon are now explicitly restricting this practice. If platform-level enforcement tightens further (App Store-style SDK rules), BrightData's residential pool could shrink materially — which would matter to anyone purchasing residential proxies from them. For Novada, this is a secondary supply risk: residential proxy quality/availability from BrightData resellers could degrade over the next 12–18 months. It is also an opportunity: ethical, directly-consented IP sourcing becomes a differentiator as regulatory scrutiny increases.
