## AI Insights

1. **MCP server cadence is becoming a competitive moat.** Firecrawl refreshed their MCP server on May 3 (2 days ago), BrightData runs an MCP Playground, and Apify is pushing the Skills SEP-2640 standard forward. First-class MCP support is no longer a differentiator — it is table stakes. The new differentiation axis is *iteration velocity*: vendors shipping MCP updates weekly will win developer mindshare over those updating quarterly.

2. **The proxy IP arms race is giving way to AI-assisted bypass.** Oxylabs' AI Web Unblocker (−43 % block rates), BrightData's targeted social-media scrapers, and Decodo's JS-site Universal Scraper all tackle the same structural problem: anti-bot defenses are sophisticated enough that raw IP pool size no longer wins. The next competitive axis is AI-driven behavioral mimicry and adaptive fingerprinting — not megapool headcount.

3. **Exa's $85 M raise accelerates the commoditization of keyword search for AI.** With neural search achieving sub-200 ms latency and Nebius bundling Tavily into its AI cloud, market expectations are shifting: results for AI agents should be semantically ranked, real-time, and cheap. Pure keyword or proxy-based retrieval products face structural margin compression unless they add semantic reranking — or get acquired before the window closes.
