## AI Insights

### 1. Nebius is now a hyperscaler — Tavily is no longer just a search API
Nebius's Q1 print ($399M, +684% YoY) and the $27B Meta megadeal position Nebius as a genuine AI infrastructure hyperscaler alongside AWS, Azure, and GCP. For Novada, this means Tavily is now backed by nearly unlimited compute and capital — it can afford to undercut on price, over-invest in reliability, and bundle search as a free feature inside a broader Nebius cloud stack. Competing against Tavily head-on on search quality alone is insufficient; the strategic response must focus on differentiation (speed, vertical-specific data, or OSS trust) rather than feature parity.

### 2. MCP is the new plugin store — and Cursor/Claude Code are the gatekeepers
Five of eight competitors (Firecrawl, BrightData, Exa, Apify, Tavily via NVIDIA) now have active MCP presence. Firecrawl's May 12 update to become an official Cursor Marketplace plugin — with one-click install — is the equivalent of securing App Store placement. The SSE-to-Streamable-HTTP migration across the industry (Apify done, MCP spec v4 live) shows the protocol is stabilising. Novada's own MCP surface should be audited now: if we are not in the Cursor Marketplace and Claude Docker MCP Catalog before summer 2026, acquiring developer mindshare will require paid promotion to overcome the default discoverability gap.

### 3. The proxy market is mid-consolidation: pricing floors rising, SMB getting squeezed
Oxylabs removed PAYG in March, BrightData dropped new-customer mobile proxies in April, and Decodo is competing on "best value" as the explicit SMB alternative. The pattern is clear — proxy incumbents are pruning low-margin, high-churn tiers and pushing enterprise contracts. For any Novada product that uses residential or mobile proxies, now is the time to re-evaluate vendor lock-in risk, negotiate multi-year rates before further price hikes, or accelerate toward alternative data acquisition methods (direct crawl, MCP-based real-time access) that bypass the proxy layer entirely.
