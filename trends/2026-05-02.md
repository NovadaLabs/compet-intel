## Novada Strategic Signals

1. **Package web-data APIs as MCP servers** — Versium, GoodData, and Flexmls prove vertical data providers are doing this now; first-mover advantage is narrowing.
2. **Proxy layer for agent browsers** — "Hidden Dangers" paper + MCP-B discussion confirm agents need sandboxed, authenticated proxy infrastructure; Novada's proxy products are directly applicable.
3. **Structured data API differentiation** — As Firecrawl/Crawl4AI commoditise raw HTML extraction, the moat shifts to structured, schema-validated data with guaranteed freshness — position here.
4. **n8n + CrewAI integrations** — Both support MCP natively and are top-5 GitHub trending; building official Novada nodes/tools here would accelerate distribution.
