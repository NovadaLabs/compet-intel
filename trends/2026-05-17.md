## Top 3 Signals for Novada

### 1. MCP Goes Enterprise — AWS GA + TikTok Ads Launch
AWS made its MCP Server **generally available on May 6, 2026**, giving AI coding agents secure, auditable access to any AWS API through a single tool interface. One week later (May 13), **TikTok opened its Ads platform via MCP**, letting agents plan, launch and optimise campaigns without manual intervention. The MCP ecosystem now counts **10,000+ public servers** governed by the Linux Foundation's Agentic AI Foundation.

**Novada relevance:** Every major platform is building a first-class MCP surface. Novada's proxy and web-data APIs are the natural routing layer between agent orchestrators and these MCP endpoints — especially where rate limits, auth rotation, and IP diversity are needed.

### 2. Claude Gets Agentic Memory & Orchestration
Anthropic shipped three new **Claude Managed Agents** capabilities (May 6–7): **Dreaming** (cross-session memory & self-improvement), **Outcomes** (goal-rubric-driven agents), and **Multiagent Orchestration** (lead agent delegates to specialists). Claude Code gained scheduled runs (hourly/nightly/weekly). Anthropic also launched **Claude for Small Business** (May 13) with 15 prebuilt workflows and native integrations with QuickBooks, HubSpot, Google Workspace, and more.

**Novada relevance:** Long-running, memory-aware agents will need persistent, reliable web-data pipelines. Novada's API should be positioned as the stable data layer beneath these autonomous loops.

### 3. "Hidden Dangers of Browsing Agents" — Security Gap Emerges
ArXiv paper **2505.13076** (May 2026) presents the first end-to-end threat model for browsing AI agents, identifying systemic vulnerabilities across multiple architectural layers. HN's **AgentSecrets** (zero-knowledge credential proxy) and the **WebMCP** proposal reflect developer anxiety around agent credential exposure.

**Novada relevance:** A credentialed, auditable proxy layer that abstracts secrets from agents is no longer a nice-to-have — it's a security requirement. Novada's proxy infrastructure maps directly to this need.
