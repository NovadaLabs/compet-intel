## Novada Strategic Signals

1. **MCP endpoint** — Launch an official Novada MCP server for proxy + scraping APIs. AWS, Azure, ServiceNow, and CrewAI have all shipped; developer expectation is hardening.
2. **Scheduled agent demand** — Claude Code scheduled routines signal persistent, high-volume crawling workloads. Prioritise bulk/batch pricing tier.
3. **Security differentiation** — 66% CVE rate in popular MCP servers creates opening for Novada to market audited, safe-by-default data access endpoints.
4. **Firecrawl watch** — Monitor their `/interact` (live browser sessions) endpoint trajectory; this is the competitive surface most likely to erode Novada's browser-automation proxy revenue.
5. **On-device agents** — Ollama at 165k stars signals local inference growth; on-device agents still need remote proxies for web access — not a threat, but a growth vector.
