## Novada Opportunity Signals

1. **AgentGateway as category signal** — The emergence of a dedicated "agentic proxy" open-source project validates the market need. Novada should publish a clear positioning brief: how Novada's proxy layer differs from and extends open-source alternatives (compliance, IP diversity, SLA, audit trail).
2. **Vision API use case** — Claude Opus 4.7's significantly improved vision means agents now process screenshots and rendered pages, not just text. Novada should document and promote a render-to-screenshot pipeline for vision-capable agents.
3. **Firecrawl commoditization response** — Free-tier autonomous scraping (Firecrawl `/agent`) is commoditizing basic crawl. Novada's differentiation must be: enterprise compliance, IP rotation at scale, CAPTCHA/bot-bypass reliability, and SLA-backed uptime — features free tiers can't offer.
4. **Authorization in multi-agent chains** — [2605.05440] shows that auth propagation in agent hierarchies is an open problem. Novada should surface credentialed proxy access (per-agent API keys, scoped permissions) as a solution narrative.
5. **$2B market milestone** — Web scraping market crossing $2B validates the space. Use in outbound sales and investor materials; pair with Novada's MCP + AI-native positioning to distinguish from legacy scraping vendors.
